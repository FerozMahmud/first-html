Learn & Explore HTML

HTML

HTML full form is HyperText Markup Language, there are two part

  1. Hypertext
  2. Markup

HTML is the code that is used to structure a web page and its content.

Tag

The HTML Tag represents the root of an HTML document.
List of basic HTML tags:

  • Anchor <.a> for link
  • Bold <.b> to make bold text
    <.strong> for bold text with emphasys
  • Body<.body> main HTML part
  • Line break <.br> for break
  • Division <.div> it is a division or part of an HTML document
  • Heading <.h1> to <.h6> for titles
  • Italic <.i> to make an italic text
  • Image <.img> for images in document
  • Ordered list <.ol> is an ordered list, Unrdered list <.ul> for an unordered list
    <.form> for form
  • List <.li> is a list item in bulleted (ordered list)
  • Paragraph <.p> for paragraph
  • Span <.span> to style part of text
  • Input <.input> for input

Here are some Self Closing Tag

  • <.br />
  • <.link />
  • <.input />
  • <.img />
  • <.hr />
